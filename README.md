Facial Emotion Detection using CNN
A deep learning project for facial emotion recognition using Convolutional Neural Networks (CNNs). This model classifies human emotions from grayscale facial images and supports real-time detection using a webcam.

>> Features
> Detects 7 emotions:
  Angry 😠
  Disgust 🤢
  Fear 😨
  Happy 😄
  Neutral 😐
  Sad 😢
  Surprise 😲
  
>> Custom CNN architecture with:
  Batch Normalization
  Dropout regularization
  L2 regularization
  
>> Performance evaluation:
  Accuracy & Loss curves
  Confusion Matrix
  Classification Report
  
Real-time emotion detection via webcam

>> Dataset
Dataset used: FER-2013
Image size: 48x48 grayscale
  Total samples:
  Train: 28,709
  Test: 7,178
